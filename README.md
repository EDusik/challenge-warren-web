## Commands

### Install Dependencies

```npm
npm install
```

### Run Unit Tests

```npm
npm run test
```
with percentage covered
```npm
npm run test -- --coverage --watchAll=false
```
### Run Cypress Tests

```npm
npx cypress open
```

### Build

```npm
npm run build
```

### Run Project

```npm
npm start
```
